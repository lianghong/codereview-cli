# Code Review CLI

LangChain-based code review tool using Claude Opus 4.5 via AWS Bedrock.

## Installation

```bash
uv venv --python 3.14
uv pip install -e .
```

## Usage

```bash
codereview /path/to/codebase
```

## Requirements

- Python 3.14+
- AWS credentials configured
- Access to AWS Bedrock Claude Opus 4.5
